# TrackForce App

## Installation and Setup Instructions

Follow the steps below to set up the TrackForce application:

### Prerequisites
- Make sure you have [Node.js](https://nodejs.org/) installed on your machine.
- Ensure that you have [npm](https://www.npmjs.com/) installed as it comes with Node.js.

### Step 1: Clone the Repository
Run the following command to clone the repository:
```bash
git clone https://github.com/Rone95/trackforce-app.git
```

### Step 2: Navigate to the Project Directory
Change your working directory to the project folder:
```bash
cd trackforce-app
```

### Step 3: Install Dependencies
Install the required dependencies by running:
```bash
npm install
```

### Step 4: Start the Application
You can start the application using:
```bash
npm start
```

### Conclusion
You should now be able to access the TrackForce app on your local machine. Visit `http://localhost:3000` in your web browser to view the application.
